Размещайте ярлыки (виджеты) на главном экране для отправки HTTP-запросов на все ваши любимые API-интерфейсы RESTful, веб-сервисы и другие ресурсы URL. Отлично подходит для проектов домашней автоматизации!

- Встроенные или пользовательские значки для ярлыков
- URL для HTTP и HTTPS
- GET, POST, PUT, DELETE, PATCH, OPTIONS, HEAD, TRACE
- Базовая и дайджест-аутентификация
- Пользовательские заголовки и тело запроса
- Отображение ответа в тосте, диалоговом окне, окне или запуск в тихом режиме
- Создание мощных рабочих процессов с помощью переменных и сценариев с использованием JavaScript
- Интеграция с Tasker и другими приложениями автоматизации, включая передачу переменных
- Импорт и экспорт в команды cURL

Некоторые идеи для чего использовать ярлыки HTTP:
- Выполнение скрипта на вашем веб-сервере
- Управление домашней системой автоматизации и устройствами IoT
- Взаимодействие с вашими проектами RaspberryPi и Arduino

Это приложение с открытым исходным кодом. Ссылка Github: https://github.com/Waboodoo/HTTP-Shortcuts. Приложение совершенно бесплатно и не содержит рекламы.
